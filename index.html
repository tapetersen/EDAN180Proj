Abstract grammar (Task 1)

  • MiniP.ast

Proposed assembly output

The stack design is normal intel i32 calling conventions but with static link passed in ecx and stored as first local variable giving a stack order of: retaddress, dynlink, statlink. The framepointer is set to the dynlink address.

Variables are allocated in prologue and temps and args as needed and deallocated after use. Register allocation is extremely primitive and loads and stores variables before/after each ususing eax ebx for doing calculations.i
